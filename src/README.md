# Source Code

This directory contains the main source code for the University Databricks Overview project.

## Structure

- `pipelines/` - Data pipeline definitions for ETL/ELT processes
- `transformations/` - Data transformation logic and business rules
- `utils/` - Shared utility functions and helpers

## Usage

Import modules from this package:

```python
from src.pipelines import your_pipeline
from src.transformations import your_transformation
from src.utils import your_utility
```
